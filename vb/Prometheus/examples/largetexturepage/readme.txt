Instructions for this demo:

1. Change the base Prometheus to load the "lgpage.maz" map and the "textures640x320.bmp"
   texture image.

2. Modify the following constants:

	Constant	Old Value	New Value
	--------------- --------------- --------------
	GRID_RES	64		128
	WALL_HEIGHT	64		128
	IMAGE_WIDTH	64		128
	IMAGE_HEIGHT	64		128
	TPAGE_XSIZE	320		640
	TPAGE_YSIZE	200		320

3. Compile and enjoy!
