Instructions for this demo:

1. Change the base Prometheus to load the "lgdefault.maz" map and the "lgtextures.bmp"
   texture image.

2. Modify the following constants:

	Constant	Old Value	New Value
	--------------- --------------- --------------
	GRID_RES	64		128
	WALL_HEIGHT	64		128
	VIEWER_HEIGHT   32		64
	IMAGE_WIDTH	64		128
	IMAGE_HEIGHT	64		128

3. Compile and enjoy!
